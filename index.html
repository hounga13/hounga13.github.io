🌟 Hounga Studio

Hounga Studio는 일상에 작은 즐거움과 편리함을 더해주는 가벼운 '스낵형 웹 애플리케이션(Snack Web App)'을 기획하고 개발하는 개인 포트폴리오 공간입니다. 복잡한 설치나 회원가입 없이 누구나 웹 브라우저만으로 즉시 접근하고 즐길 수 있는 유용한 도구들을 제공합니다.

📈 전인구 탐지기 (역발상 투자 AI 분석)

"그가 살 때 팔고, 그가 팔 때 사라!"
유명 경제 유튜버 '전인구 경제연구소'의 최신 영상 스크립트와 뉘앙스를 인공지능(AI) 기술로 분석하여, 현재 시장 상황에 대한 정반대의 투자 포지션(인버스 매매)을 위트 있게 제안하는 금융/엔터테인먼트 시뮬레이터입니다.

단순한 유머를 넘어 최신 자연어 처리 알고리즘 모델링을 사용하여 맥락을 분석하고, 주식 관련 커뮤니티에서 유행하는 '반대 매매' 밈(Meme)을 시스템화한 결과물을 산출합니다. 경제와 주식 투자를 즐기는 유저들에게 즐거운 인사이트를 제공하도록 React.js 등 최신 웹 프레임워크 기술로 개발되었습니다.

🔮 오늘 어때? (운세 스낵 콘텐츠)

현대인들에게 오늘의 운세는 단순한 미신을 넘어 하루를 준비하는 작은 활력소가 됩니다. 불확실한 무작위 데이터에 의존하지 않고, 당신만의 고유한 정보(이름, 생년월일)를 이용한 독자적인 해시 알고리즘으로 설계된 결정론적 운세 해석 엔진입니다.

이 웹앱은 금전운, 연애운, 대인운 등의 점수화된 수치와 개인 맞춤형 행운의 아이템, MBTI 매칭 데이터를 처리합니다. 사용자가 브라우저 내장 Web Share API 및 클립보드 복사를 통해 카카오톡 등 소셜 네트워크로 결과를 손쉽게 바이럴할 수 있도록 최적화된 UI/UX 및 사파리(iOS) 호환 고해상도 이미지 캡처 렌더링을 지원합니다.
